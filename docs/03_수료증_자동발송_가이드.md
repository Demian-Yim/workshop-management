# 기능 3: 수료증 자동 발송 (20명)

## 전체 플로우

```
[나노바나나프로] 수료증 템플릿 디자인
    ↓
[구글 스프레드시트] 수료자 20명 명단 관리
    ↓
[Apps Script] 수료증 생성 자동화
    ↓
┌──────────────────────────────────┐
│  방법 A: 구글 슬라이드 템플릿     │
│  → 이름/날짜 자동 삽입            │
│  → PDF 변환                     │
│  → Gmail 첨부 발송               │
├──────────────────────────────────┤
│  방법 B: 나노바나나프로 API 연동   │
│  → 템플릿에 데이터 바인딩          │
│  → PDF/이미지 자동 생성           │
│  → Gmail 첨부 발송               │
└──────────────────────────────────┘
```

## 방법 A: 구글 슬라이드 + Apps Script (무료)

### STEP 1: 구글 슬라이드로 수료증 템플릿 만들기

1. [Google Slides](https://slides.google.com) 접속
2. 빈 프레젠테이션 → 슬라이드 크기: A4 가로 (파일 → 페이지 설정)
3. 나노바나나프로에서 디자인한 수료증 배경 이미지 삽입
4. 텍스트 자리표시자 추가:
   - `{{이름}}` → 수료자 이름
   - `{{교육명}}` → 교육 과정명
   - `{{교육기간}}` → 교육 기간
   - `{{수료번호}}` → 수료 번호
   - `{{발급일}}` → 발급 날짜
5. 슬라이드 ID 메모 (URL에서 확인)

### STEP 2: 스프레드시트 설정

시트명: `수료증발송`

| A: 이름 | B: 이메일 | C: 교육명 | D: 교육기간 | E: 수료번호 | F: 발송여부 | G: 발송일시 |

**수료번호 자동생성 규칙**: `CERT-2026-001` ~ `CERT-2026-020`

### STEP 3: Apps Script 코드

```javascript
// ========================================
// 기능 3: 수료증 자동 생성 및 발송
// ========================================

const CERT_CONFIG = {
  TEMPLATE_SLIDE_ID: 'YOUR_GOOGLE_SLIDE_TEMPLATE_ID', // 수료증 템플릿 슬라이드 ID
  SHEET_NAME: '수료증발송',
  SENDER_NAME: '교육운영팀',
  EDUCATION_NAME: 'AI 활용 업무혁신 과정',
  OUTPUT_FOLDER_ID: 'YOUR_GOOGLE_DRIVE_FOLDER_ID', // 수료증 저장 폴더
  CERT_PREFIX: 'CERT-2026-',
};

/**
 * 수료증 일괄 생성 및 발송 (메인 함수)
 */
function generateAndSendCertificates() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(CERT_CONFIG.SHEET_NAME);
  const data = sheet.getDataRange().getValues();
  const folder = DriveApp.getFolderById(CERT_CONFIG.OUTPUT_FOLDER_ID);

  let successCount = 0;
  let failCount = 0;

  for (let i = 1; i < data.length; i++) {
    const name = data[i][0];
    const email = data[i][1];
    const educationName = data[i][2] || CERT_CONFIG.EDUCATION_NAME;
    const educationPeriod = data[i][3];
    const certNumber = data[i][4];
    const sentStatus = data[i][5];

    // 이미 발송된 경우 건너뛰기
    if (sentStatus === '발송완료') continue;
    if (!name || !email) continue;

    const row = i + 1;

    try {
      // 1. 수료증 PDF 생성
      const pdfFile = createCertificatePDF(name, educationName, educationPeriod, certNumber, folder);

      // 2. 이메일 발송
      sendCertificateEmail(name, email, educationName, pdfFile);

      // 3. 상태 업데이트
      sheet.getRange(row, 6).setValue('발송완료');
      sheet.getRange(row, 6).setBackground('#d9ead3');
      sheet.getRange(row, 7).setValue(new Date());

      successCount++;
      Logger.log(`[성공] ${name}(${email}) 수료증 발송 완료`);

      // Gmail 일일 한도 보호를 위한 딜레이
      Utilities.sleep(2000);

    } catch (err) {
      sheet.getRange(row, 6).setValue('발송실패');
      sheet.getRange(row, 6).setBackground('#f4cccc');
      sheet.getRange(row, 7).setValue(err.message);

      failCount++;
      Logger.log(`[실패] ${name}(${email}): ${err.message}`);
    }
  }

  // 결과 요약 알림
  const summary = `수료증 발송 완료\n성공: ${successCount}명\n실패: ${failCount}명`;
  Logger.log(summary);

  // 관리자에게 결과 이메일 발송
  GmailApp.sendEmail(
    Session.getActiveUser().getEmail(),
    `[수료증 발송 결과] ${successCount}명 성공, ${failCount}명 실패`,
    summary
  );
}

/**
 * 수료증 PDF 생성
 */
function createCertificatePDF(name, educationName, educationPeriod, certNumber, folder) {
  // 1. 템플릿 슬라이드 복사
  const templateFile = DriveApp.getFileById(CERT_CONFIG.TEMPLATE_SLIDE_ID);
  const copyFile = templateFile.makeCopy(`수료증_${name}`, folder);
  const copyId = copyFile.getId();

  // 2. 텍스트 치환
  const presentation = SlidesApp.openById(copyId);
  const slide = presentation.getSlides()[0];

  const replacements = {
    '{{이름}}': name,
    '{{교육명}}': educationName,
    '{{교육기간}}': educationPeriod,
    '{{수료번호}}': certNumber,
    '{{발급일}}': Utilities.formatDate(new Date(), 'Asia/Seoul', 'yyyy년 MM월 dd일'),
  };

  // 모든 텍스트 요소에서 치환
  slide.getShapes().forEach(shape => {
    if (shape.getText) {
      const textRange = shape.getText();
      Object.keys(replacements).forEach(key => {
        textRange.replaceAllText(key, replacements[key]);
      });
    }
  });

  presentation.saveAndClose();

  // 3. PDF로 변환
  const pdfBlob = DriveApp.getFileById(copyId).getAs('application/pdf');
  pdfBlob.setName(`수료증_${name}_${certNumber}.pdf`);

  const pdfFile = folder.createFile(pdfBlob);

  // 4. 임시 슬라이드 파일 삭제
  DriveApp.getFileById(copyId).setTrashed(true);

  return pdfFile;
}

/**
 * 수료번호 자동 생성 (유틸리티)
 */
function generateCertNumbers() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(CERT_CONFIG.SHEET_NAME);
  const data = sheet.getDataRange().getValues();

  for (let i = 1; i < data.length; i++) {
    if (!data[i][4]) { // 수료번호가 없으면 생성
      const num = String(i).padStart(3, '0');
      sheet.getRange(i + 1, 5).setValue(CERT_CONFIG.CERT_PREFIX + num);
    }
  }
}

/**
 * 테스트: 1명에게만 수료증 발송
 */
function testSendOneCertificate() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(CERT_CONFIG.SHEET_NAME);
  const name = sheet.getRange(2, 1).getValue();
  const email = sheet.getRange(2, 2).getValue();
  const educationName = sheet.getRange(2, 3).getValue() || CERT_CONFIG.EDUCATION_NAME;
  const educationPeriod = sheet.getRange(2, 4).getValue();
  const certNumber = sheet.getRange(2, 5).getValue();

  const folder = DriveApp.getFolderById(CERT_CONFIG.OUTPUT_FOLDER_ID);
  const pdfFile = createCertificatePDF(name, educationName, educationPeriod, certNumber, folder);
  sendCertificateEmail(name, email, educationName, pdfFile);

  Logger.log(`테스트 발송 완료: ${name}(${email})`);
}
```

## 방법 B: 나노바나나프로 활용 (고품질)

### STEP 1: 나노바나나프로에서 수료증 디자인

1. 나노바나나프로 접속
2. 수료증 템플릿 선택 또는 새로 디자인
3. 다음 요소 포함:
   - 교육기관 로고
   - 수료증 제목 ("수료증" 또는 "Certificate of Completion")
   - 수료자 이름 영역
   - 교육 과정명
   - 교육 기간
   - 수료번호
   - 발급일
   - 서명/직인 영역
   - 장식 테두리/엠블럼
4. PNG 또는 PDF로 내보내기 (배경 이미지로 사용)

### STEP 2: 구글 슬라이드에 배경으로 적용

1. 나노바나나프로에서 만든 디자인을 구글 슬라이드 배경으로 삽입
2. 텍스트 자리표시자({{이름}} 등)를 정확한 위치에 배치
3. 폰트, 크기, 색상 조정

### 나노바나나프로 디자인 팁

- **수료증 스타일**: 격식 있는 세리프 폰트 + 금색/남색 조합
- **레이아웃**: 중앙 정렬, 상하 여백 충분히
- **필수 요소**: 기관명, 수료증 타이틀, 이름, 과정명, 기간, 번호, 날짜, 직인
- **해상도**: 300 DPI 이상 (인쇄 품질)
- **파일 형식**: PNG (투명 배경) 또는 고해상도 PDF
