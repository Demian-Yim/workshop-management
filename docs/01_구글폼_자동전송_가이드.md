# 기능 1: 구글폼 접수 시 이메일 + 문자 자동 전송

## 전체 플로우

```
수강생이 구글 폼 작성
    ↓
구글 스프레드시트에 자동 기록
    ↓
Apps Script 트리거 실행
    ↓
┌─────────────────┬──────────────────┐
│  Gmail로         │  문자 API로       │
│  확인 이메일 발송  │  SMS 발송         │
└─────────────────┴──────────────────┘
    ↓
스프레드시트에 발송 상태 기록
```

## STEP 1: 구글 폼 만들기

1. [Google Forms](https://forms.google.com) 접속
2. 새 양식 만들기
3. 필수 항목 추가:
   - 이름 (단답형)
   - 이메일 (단답형, 이메일 유효성 검사)
   - 전화번호 (단답형)
   - 소속/직위 (단답형)
   - 교육 과정 선택 (객관식)
4. **응답** 탭 → **스프레드시트에 연결** 클릭

## STEP 2: 스프레드시트 설정

연결된 스프레드시트에서:
- 시트 이름을 `접수현황`으로 변경
- 마지막 열에 `이메일발송`, `문자발송` 컬럼 추가

## STEP 3: Apps Script 코드 작성

스프레드시트에서 **확장 프로그램 → Apps Script** 클릭 후 아래 코드 입력:

```javascript
// ========================================
// 기능 1: 구글폼 접수 → 이메일 + 문자 자동 발송
// ========================================

// 설정값
const CONFIG = {
  SENDER_NAME: '교육운영팀',
  EDUCATION_NAME: 'AI 활용 업무혁신 과정',
  EDUCATION_DATE: '2026년 3월 15일~16일',
  EDUCATION_LOCATION: '서울 강남구 OO빌딩 5층',

  // 문자 API 설정 (Aligo 기준)
  SMS_API_KEY: 'YOUR_API_KEY',
  SMS_USER_ID: 'YOUR_USER_ID',
  SMS_SENDER: '010-0000-0000', // 발신번호 (사전 등록 필요)
};

/**
 * 폼 제출 시 자동 실행되는 함수
 * 트리거 설정: 폼 제출 이벤트
 */
function onFormSubmit(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('접수현황');
  const lastRow = sheet.getLastRow();

  // 폼 응답 데이터 가져오기
  const timestamp = sheet.getRange(lastRow, 1).getValue();
  const name = sheet.getRange(lastRow, 2).getValue();
  const email = sheet.getRange(lastRow, 3).getValue();
  const phone = sheet.getRange(lastRow, 4).getValue();
  const company = sheet.getRange(lastRow, 5).getValue();
  const course = sheet.getRange(lastRow, 6).getValue();

  // 이메일 발송
  try {
    sendConfirmationEmail(name, email, company, course);
    sheet.getRange(lastRow, 7).setValue('발송완료'); // 이메일 발송 상태
    sheet.getRange(lastRow, 7).setBackground('#d9ead3');
  } catch (err) {
    sheet.getRange(lastRow, 7).setValue('발송실패: ' + err.message);
    sheet.getRange(lastRow, 7).setBackground('#f4cccc');
  }

  // 문자 발송
  try {
    sendSMS(name, phone, course);
    sheet.getRange(lastRow, 8).setValue('발송완료'); // 문자 발송 상태
    sheet.getRange(lastRow, 8).setBackground('#d9ead3');
  } catch (err) {
    sheet.getRange(lastRow, 8).setValue('발송실패: ' + err.message);
    sheet.getRange(lastRow, 8).setBackground('#f4cccc');
  }
}

/**
 * 확인 이메일 발송
 */

/**
 * SMS 문자 발송 (Aligo API 기준)
 */
function sendSMS(name, phone, course) {
  const message = `[${CONFIG.SENDER_NAME}] ${name}님, ${CONFIG.EDUCATION_NAME} 접수가 완료되었습니다. 교육일: ${CONFIG.EDUCATION_DATE} 장소: ${CONFIG.EDUCATION_LOCATION}`;

  // Aligo API 호출
  const url = 'https://apis.aligo.in/send/';
  const payload = {
    'key': CONFIG.SMS_API_KEY,
    'userid': CONFIG.SMS_USER_ID,
    'sender': CONFIG.SMS_SENDER,
    'receiver': phone.replace(/-/g, ''),
    'msg': message,
    'msg_type': message.length > 90 ? 'LMS' : 'SMS',
  };

  const options = {
    'method': 'post',
    'payload': payload,
    'muteHttpExceptions': true
  };

  const response = UrlFetchApp.fetch(url, options);
  const result = JSON.parse(response.getContentText());

  if (result.result_code !== '1') {
    throw new Error(result.message || '문자 발송 실패');
  }
}

// ========================================
// 트리거 설정 함수 (최초 1회 실행)
// ========================================
function setupFormTrigger() {
  // 기존 트리거 삭제
  const triggers = ScriptApp.getProjectTriggers();
  triggers.forEach(trigger => ScriptApp.deleteTrigger(trigger));

  // 폼 제출 트리거 등록
  ScriptApp.newTrigger('onFormSubmit')
    .forSpreadsheet(SpreadsheetApp.getActiveSpreadsheet())
    .onFormSubmit()
    .create();

  Logger.log('폼 제출 트리거가 설정되었습니다.');
}
```

## STEP 4: 트리거 설정

1. Apps Script 에디터에서 `setupFormTrigger` 함수 선택
2. **실행** 클릭
3. 권한 승인 (Gmail, 외부 URL 접근 등)

## STEP 5: 문자 API 설정 (Aligo 기준)

1. [Aligo](https://smartsms.aligo.in) 회원가입
2. API 키 발급
3. 발신번호 등록 (본인인증 필요)
4. CONFIG에 API 키, 사용자 ID, 발신번호 입력
