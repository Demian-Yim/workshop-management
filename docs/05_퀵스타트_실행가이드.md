# 퀵스타트 실행 가이드

## 전체 실행 순서 (약 2~3시간)

---

### Phase 1: 기반 설정 (30분)

#### 1-1. 구글 스프레드시트 생성
- [ ] 새 스프레드시트 생성: "교육운영 자동화 대시보드"
- [ ] 시트1: `접수현황` (이름, 이메일, 전화번호, 소속, 과정, 이메일발송, 문자발송)
- [ ] 시트2: `강사서류관리` (강사명~상태까지 11개 컬럼)
- [ ] 시트3: `수료증발송` (이름~발송일시까지 7개 컬럼)

#### 1-2. 구글 폼 생성
- [ ] 수강 접수 폼 → `접수현황` 시트에 연결
- [ ] 강사 서류 제출 폼 (파일 업로드 포함) → 별도 시트에 연결

#### 1-3. 문자 API 가입
- [ ] Aligo(smartsms.aligo.in) 또는 CoolSMS 가입
- [ ] API 키 발급
- [ ] 발신번호 등록

---

### Phase 2: 기능 1 구축 - 접수 자동 전송 (30분)

- [ ] 스프레드시트 → 확장 프로그램 → Apps Script
- [ ] `01_구글폼_자동전송_가이드.md`의 코드 입력
- [ ] CONFIG 설정값 수정 (교육명, 날짜, 장소, API 키)
- [ ] `setupFormTrigger()` 실행
- [ ] 권한 승인
- [ ] 테스트: 구글 폼 제출 → 이메일/문자 수신 확인

---

### Phase 3: 기능 2 구축 - 강사 서류 수집 (30분)

- [ ] 같은 Apps Script에 기능 2 코드 추가
- [ ] DOC_CONFIG 설정값 수정 (폼 URL, 마감일 등)
- [ ] `setupDocTriggers()` 실행 (매일 9시 체크)
- [ ] 강사 정보 입력 (시트2에 강사명, 이메일 등)
- [ ] `sendDocumentRequest()` 실행하여 서류 요청 이메일 발송
- [ ] 테스트: 서류 제출 → 상태 업데이트 확인

---

### Phase 4: 기능 3 구축 - 수료증 발송 (40분)

#### 4-1. 수료증 디자인
- [ ] 나노바나나프로에서 수료증 템플릿 디자인
- [ ] PNG/PDF로 내보내기

#### 4-2. 구글 슬라이드 템플릿
- [ ] 새 구글 슬라이드 생성 (A4 가로)
- [ ] 나노바나나프로 디자인을 배경으로 삽입
- [ ] 텍스트 자리표시자 배치: {{이름}}, {{교육명}}, {{교육기간}}, {{수료번호}}, {{발급일}}
- [ ] 슬라이드 ID 복사

#### 4-3. 수료증 발송 코드
- [ ] Apps Script에 기능 3 코드 추가
- [ ] CERT_CONFIG 설정값 수정 (슬라이드 ID, 폴더 ID)
- [ ] 수료증 저장용 구글 드라이브 폴더 생성
- [ ] 수료자 20명 명단 입력 (시트3)
- [ ] `generateCertNumbers()` 실행하여 수료번호 생성
- [ ] `testSendOneCertificate()` 실행하여 1명 테스트
- [ ] `generateAndSendCertificates()` 실행하여 전체 발송

---

### Phase 5: Google AI Studio 활용 (20분)

- [ ] Google AI Studio 접속
- [ ] `04_Google_AI_Studio_프롬프트.md`에서 필요한 프롬프트 적용

## 문제 해결

| 문제 | 해결 방법 |
|------|-----------|
| 이메일이 안 보내짐 | Apps Script 권한 재승인, Gmail 일일 한도(500통) 확인 |
| 문자가 안 감 | API 키 확인, 발신번호 등록 여부 확인, 잔액 확인 |
| 트리거가 안 됨 | Apps Script → 트리거 메뉴에서 활성화 상태 확인 |
| 수료증 PDF 깨짐 | 구글 슬라이드 폰트 확인, 텍스트 영역 크기 조정 |
| 독촉 메일 중복 | 스프레드시트 독촉횟수 컬럼 확인, 시간대 설정 확인 |
