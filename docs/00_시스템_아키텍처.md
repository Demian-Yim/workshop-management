# 기업교육 운영 자동화 시스템 아키텍처

## 전체 시스템 구성도

```
[수강생/강사]
    |
    v
[구글 폼] ──> [구글 스프레드시트] ──> [Apps Script]
                                         |
                    ┌────────────────────┼────────────────────┐
                    v                    v                    v
              [Gmail API]        [문자 API]          [나노바나나프로]
              이메일 발송         SMS 발송            수료증 디자인
                    |                    |                    |
                    v                    v                    v
              수강 확인 메일       접수 알림 문자       수료증 PDF
              독촉 메일                                      |
              서류 요청 메일                                  v
                                                      [Gmail 발송]
```

## 3가지 핵심 기능

### 기능 1: 구글폼 접수 → 자동 이메일 + 문자
- **트리거**: 구글 폼 제출 이벤트
- **도구**: Google Apps Script + Gmail + 문자 API (Aligo/CoolSMS)
- **플로우**: 폼 제출 → 스프레드시트 기록 → 확인 이메일 발송 → SMS 발송

### 기능 2: 강사 서류 수집 자동화 + 독촉 메일
- **트리거**: 시간 기반 트리거 (매일 오전 9시 체크)
- **도구**: Google Apps Script + Gmail + 구글 드라이브
- **플로우**: 서류 요청 메일 → 제출 여부 추적 → 미제출시 독촉 메일 자동 발송

### 기능 3: 수료증 자동 발송 (20명)
- **트리거**: 수동 실행 또는 교육 종료일 자동 실행
- **도구**: 나노바나나프로(수료증 디자인) + Apps Script + Gmail
- **플로우**: 수료증 템플릿 생성 → 이름 자동 삽입 → PDF 변환 → 이메일 발송

## 사용 도구 목록

| 도구 | 용도 | 비용 |
|------|------|------|
| 구글 폼 | 수강 접수, 서류 제출 | 무료 |
| 구글 스프레드시트 | 데이터 관리 | 무료 |
| Google Apps Script | 자동화 엔진 | 무료 |
| Gmail | 이메일 발송 | 무료 (일 500통) |
| Aligo/CoolSMS | 문자 발송 | 건당 9~15원 |
| 나노바나나프로 | 수료증 디자인 | 유료 구독 |
| Google AI Studio | AI 프롬프트 활용 | 무료 |

## 구글 스프레드시트 구조

### 시트1: 수강생 명단
| 이름 | 이메일 | 전화번호 | 소속 | 접수일시 | 이메일발송여부 | 문자발송여부 |

### 시트2: 강사 서류 관리
| 강사명 | 이메일 | 프로필 | 이력서 | 강의계획서 | 서류요청일 | 제출완료일 | 독촉횟수 |

### 시트3: 수료증 발송
| 이름 | 이메일 | 교육명 | 교육기간 | 수료번호 | 수료증발송여부 | 발송일시 |
