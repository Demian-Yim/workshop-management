# 기능 2: 강사 서류 수집 자동화 + 독촉 메일 자동 발송

## 전체 플로우

```
[관리자] 강사 정보 입력 (스프레드시트)
    ↓
Apps Script 실행 → 서류 요청 이메일 발송
    ↓
[강사] 구글 폼으로 서류 제출 (프로필, 이력서, 강의계획서)
    ↓
구글 드라이브에 자동 저장 + 스프레드시트 상태 업데이트
    ↓
매일 오전 9시 자동 체크
    ↓
미제출 강사에게 독촉 메일 자동 발송
    (1차: 3일 후 / 2차: 5일 후 / 3차: 7일 후)
```

## STEP 1: 스프레드시트 설정

### 시트명: `강사서류관리`

| A: 강사명 | B: 이메일 | C: 전화번호 | D: 프로필사진 | E: 이력서 | F: 강의계획서 | G: 요청일 | H: 마감일 | I: 제출완료일 | J: 독촉횟수 | K: 상태 |
|-----------|-----------|-------------|--------------|-----------|--------------|-----------|-----------|-------------|-------------|---------|

**상태값**: 요청전 / 요청완료 / 부분제출 / 제출완료 / 마감초과

## STEP 2: 서류 제출용 구글 폼 만들기

1. 새 구글 폼 생성: "강사 서류 제출"
2. 항목:
   - 강사명 (단답형)
   - 이메일 (단답형)
   - 프로필 사진 (파일 업로드 - 이미지만)
   - 이력서 (파일 업로드 - PDF, 워드)
   - 강의계획서 (파일 업로드 - PDF, 워드, PPT)
3. 응답을 스프레드시트에 연결

## STEP 3: Apps Script 코드

```javascript
// ========================================
// 기능 2: 강사 서류 수집 자동화 + 독촉 메일
// ========================================

const DOC_CONFIG = {
  SHEET_NAME: '강사서류관리',
  SENDER_NAME: '교육운영팀',
  EDUCATION_NAME: 'AI 활용 업무혁신 과정',
  FORM_URL: 'https://forms.gle/YOUR_FORM_LINK', // 서류 제출 폼 URL
  DEADLINE_DAYS: 7,        // 마감까지 일수
  REMIND_INTERVALS: [3, 5, 7], // 독촉 메일 발송 간격 (요청일 기준 일수)
  MAX_REMINDERS: 3,        // 최대 독촉 횟수
};

/**
 * 전체 강사에게 서류 요청 이메일 발송 (수동 실행)
 */
function sendDocumentRequest() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(DOC_CONFIG.SHEET_NAME);
  const data = sheet.getDataRange().getValues();

  for (let i = 1; i < data.length; i++) { // 헤더 제외
    const name = data[i][0];
    const email = data[i][1];
    const status = data[i][10]; // K열: 상태

    if (!email || status === '제출완료') continue;

    try {
      sendRequestEmail(name, email);

      // 상태 업데이트
      const row = i + 1;
      sheet.getRange(row, 7).setValue(new Date()); // 요청일
      sheet.getRange(row, 8).setValue(getDeadlineDate()); // 마감일
      sheet.getRange(row, 10).setValue(0); // 독촉횟수 초기화
      sheet.getRange(row, 11).setValue('요청완료');
      sheet.getRange(row, 11).setBackground('#c9daf8');

    } catch (err) {
      Logger.log(`[오류] ${name}(${email}): ${err.message}`);
    }
  }
}

/**
 * 서류 요청 이메일 발송
 */
function sendRequestEmail(name, email) {
  const subject = `[서류 제출 요청] ${DOC_CONFIG.EDUCATION_NAME} 강사 서류 안내`;

  const htmlBody = `
    <div style="font-family: 'Malgun Gothic', Arial, sans-serif; max-width: 600px; margin: 0 auto; padding: 20px;">
      <div style="background: linear-gradient(135deg, #11998e 0%, #38ef7d 100%); padding: 30px; border-radius: 10px 10px 0 0;">
        <h1 style="color: white; margin: 0; font-size: 22px;">강사 서류 제출 요청</h1>
      </div>

      <div style="background: white; padding: 30px; border: 1px solid #e0e0e0; border-top: none;">
        <p style="font-size: 16px; color: #333;">안녕하세요, <strong>${name}</strong> 강사님!</p>
        <p style="color: #555; line-height: 1.8;">
          ${DOC_CONFIG.EDUCATION_NAME} 강의 준비를 위해<br>
          아래 서류를 제출해 주시기 바랍니다.
        </p>

        <div style="background: #f0f7ff; padding: 20px; border-radius: 8px; margin: 20px 0;">
          <h3 style="margin: 0 0 15px 0; color: #1a73e8;">제출 서류 목록</h3>
          <ol style="color: #333; line-height: 2;">
            <li><strong>프로필 사진</strong> (증명사진 또는 강의 사진)</li>
            <li><strong>이력서</strong> (PDF 또는 워드 형식)</li>
            <li><strong>강의계획서</strong> (PDF, 워드, PPT)</li>
          </ol>
        </div>

        <div style="background: #fff3cd; padding: 15px; border-radius: 8px; border-left: 4px solid #ffc107; margin: 20px 0;">
          <p style="margin: 0; color: #856404;">
            <strong>제출 마감일: ${Utilities.formatDate(getDeadlineDate(), 'Asia/Seoul', 'yyyy년 MM월 dd일')}</strong>
          </p>
        </div>

        <div style="text-align: center; margin: 30px 0;">
          <a href="${DOC_CONFIG.FORM_URL}"
             style="background: #1a73e8; color: white; padding: 15px 40px;
                    border-radius: 8px; text-decoration: none; font-size: 16px;
                    font-weight: bold; display: inline-block;">
            서류 제출하기
          </a>
        </div>

        <p style="color: #999; font-size: 12px; text-align: center;">
          문의: ${DOC_CONFIG.SENDER_NAME} | edu@company.com
        </p>
      </div>
    </div>
  `;

  GmailApp.sendEmail(email, subject, '', {
    htmlBody: htmlBody,
    name: DOC_CONFIG.SENDER_NAME
  });
}

/**
 * 독촉 메일 체크 및 발송 (매일 오전 9시 자동 실행)
 */
function checkAndSendReminders() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(DOC_CONFIG.SHEET_NAME);
  const data = sheet.getDataRange().getValues();
  const today = new Date();

  for (let i = 1; i < data.length; i++) {
    const name = data[i][0];
    const email = data[i][1];
    const requestDate = data[i][6]; // 요청일
    const completionDate = data[i][8]; // 제출완료일
    const reminderCount = data[i][9] || 0; // 독촉횟수
    const status = data[i][10]; // 상태

    // 이미 제출 완료했거나, 요청 전이면 건너뛰기
    if (status === '제출완료' || status === '요청전' || !requestDate) continue;

    // 최대 독촉 횟수 초과하면 건너뛰기
    if (reminderCount >= DOC_CONFIG.MAX_REMINDERS) continue;

    // 요청일로부터 경과 일수 계산
    const daysSinceRequest = Math.floor((today - requestDate) / (1000 * 60 * 60 * 24));

    // 독촉 메일 발송 시점인지 확인
    const nextRemindDay = DOC_CONFIG.REMIND_INTERVALS[reminderCount];
    if (daysSinceRequest >= nextRemindDay) {
      try {
        sendReminderEmail(name, email, reminderCount + 1, data[i]);

        const row = i + 1;
        sheet.getRange(row, 10).setValue(reminderCount + 1);

        // 마감 초과 여부 체크
        if (daysSinceRequest > DOC_CONFIG.DEADLINE_DAYS) {
          sheet.getRange(row, 11).setValue('마감초과');
          sheet.getRange(row, 11).setBackground('#f4cccc');
        }

        Logger.log(`[독촉 ${reminderCount + 1}차] ${name}(${email}) 발송 완료`);
      } catch (err) {
        Logger.log(`[오류] ${name}: ${err.message}`);
      }
    }
  }
}

/**
 * 독촉 이메일 발송
 */
function sendReminderEmail(name, email, count, rowData) {
  const urgencyColors = {
    1: { bg: '#fff3cd', border: '#ffc107', text: '#856404', label: '1차 안내' },
    2: { bg: '#ffe0cc', border: '#ff8c00', text: '#cc5500', label: '2차 독촉' },
    3: { bg: '#f8d7da', border: '#dc3545', text: '#721c24', label: '최종 독촉' },
  };

  const style = urgencyColors[count] || urgencyColors[1];

  // 미제출 서류 파악
  const missing = [];
  if (!rowData[3]) missing.push('프로필 사진');
  if (!rowData[4]) missing.push('이력서');
  if (!rowData[5]) missing.push('강의계획서');

  const subject = `[${style.label}] ${DOC_CONFIG.EDUCATION_NAME} 서류 제출 요청 - ${name} 강사님`;

  const htmlBody = `
    <div style="font-family: 'Malgun Gothic', Arial, sans-serif; max-width: 600px; margin: 0 auto; padding: 20px;">
      <div style="background: ${style.bg}; padding: 20px; border-radius: 10px; border-left: 5px solid ${style.border};">
        <h2 style="color: ${style.text}; margin: 0;">${style.label}: 서류 제출 요청</h2>
      </div>

      <div style="background: white; padding: 30px; border: 1px solid #e0e0e0; margin-top: -1px;">
        <p><strong>${name}</strong> 강사님, 안녕하세요.</p>
        <p>아직 아래 서류가 제출되지 않았습니다.</p>

        <div style="background: #f8f9fa; padding: 15px; border-radius: 8px; margin: 15px 0;">
          <h4 style="margin: 0 0 10px 0; color: #dc3545;">미제출 서류</h4>
          <ul style="margin: 0; padding-left: 20px;">
            ${missing.map(doc => `<li style="color: #333; padding: 3px 0;"><strong>${doc}</strong></li>`).join('')}
          </ul>
        </div>

        ${count === 3 ? `
        <div style="background: #f8d7da; padding: 15px; border-radius: 8px; margin: 15px 0;">
          <p style="margin: 0; color: #721c24; font-weight: bold;">
            이 메일은 최종 독촉입니다. 빠른 제출을 부탁드립니다.
          </p>
        </div>
        ` : ''}

        <div style="text-align: center; margin: 25px 0;">
          <a href="${DOC_CONFIG.FORM_URL}"
             style="background: ${style.border}; color: white; padding: 14px 35px;
                    border-radius: 8px; text-decoration: none; font-weight: bold;">
            지금 서류 제출하기
          </a>
        </div>
      </div>
    </div>
  `;

  GmailApp.sendEmail(email, subject, '', {
    htmlBody: htmlBody,
    name: DOC_CONFIG.SENDER_NAME
  });
}

/**
 * 서류 제출 시 스프레드시트 업데이트 (폼 제출 트리거)
 */
function onDocFormSubmit(e) {
  const responseSheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('서류제출응답');
  const managementSheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(DOC_CONFIG.SHEET_NAME);

  const lastRow = responseSheet.getLastRow();
  const submittedEmail = responseSheet.getRange(lastRow, 3).getValue(); // 이메일 열

  // 강사서류관리 시트에서 해당 강사 찾기
  const data = managementSheet.getDataRange().getValues();
  for (let i = 1; i < data.length; i++) {
    if (data[i][1] === submittedEmail) {
      const row = i + 1;

      // 제출된 서류 체크 (파일 업로드 항목)
      const profileUrl = responseSheet.getRange(lastRow, 4).getValue();
      const resumeUrl = responseSheet.getRange(lastRow, 5).getValue();
      const planUrl = responseSheet.getRange(lastRow, 6).getValue();

      if (profileUrl) managementSheet.getRange(row, 4).setValue(profileUrl);
      if (resumeUrl) managementSheet.getRange(row, 5).setValue(resumeUrl);
      if (planUrl) managementSheet.getRange(row, 6).setValue(planUrl);

      // 상태 업데이트
      const allSubmitted = profileUrl && resumeUrl && planUrl;
      if (allSubmitted) {
        managementSheet.getRange(row, 9).setValue(new Date()); // 제출완료일
        managementSheet.getRange(row, 11).setValue('제출완료');
        managementSheet.getRange(row, 11).setBackground('#d9ead3');
      } else {
        managementSheet.getRange(row, 11).setValue('부분제출');
        managementSheet.getRange(row, 11).setBackground('#fce5cd');
      }

      break;
    }
  }
}

// ========================================
// 트리거 설정 (최초 1회 실행)
// ========================================
function setupDocTriggers() {
  // 매일 오전 9시 독촉 메일 체크
  ScriptApp.newTrigger('checkAndSendReminders')
    .timeBased()
    .atHour(9)
    .everyDays(1)
    .create();

  Logger.log('독촉 메일 자동 체크 트리거가 설정되었습니다.');
}

// 유틸리티 함수
function getDeadlineDate() {
  const deadline = new Date();
  deadline.setDate(deadline.getDate() + DOC_CONFIG.DEADLINE_DAYS);
  return deadline;
}
```

## STEP 4: 트리거 설정

1. `setupDocTriggers()` 실행 → 매일 오전 9시 자동 체크
2. 서류 제출 구글 폼의 응답 스프레드시트에서도 `onDocFormSubmit` 트리거 설정
