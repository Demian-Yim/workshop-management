## 앱 소개

워크샵 운영앱은 오프라인/온라인 교육 워크샵을 실시간으로 운영하기 위한 통합 플랫폼입니다.

### 주요 특징

| 기능 | 설명 |
| --- | --- |
| 실시간 데이터 | 출석, 게시글, 투표 등 모든 데이터가 실시간 동기화됩니다 |
| 세션 코드 | 4자리 숫자 코드로 간편하게 세션에 참여합니다 |
| 3가지 역할 | 학습자 / 강사 / 관리자 각각 맞춤 화면을 제공합니다 |
| 모바일 최적화 | 학습자 화면은 스마트폰에 최적화되어 있습니다 |
| QR 체크인 | QR 코드 스캔으로 세션 참여와 출석을 한 번에 완료합니다 |
| 셀카 출석 | 카메라로 셀카를 촬영하여 출석을 등록합니다 |
| AI 캐릭터 생성 | 셀카 기반으로 나만의 캐릭터 아바타를 자동 생성합니다 (카툰/애니메/픽셀) |
| 키워드 자기소개 | 키워드를 선택하면 자기소개가 자동으로 생성됩니다 |

---

## 역할 안내

앱 메인 화면(`/`)에서 3가지 역할 중 하나를 선택하여 시작합니다.

| 역할 | 대상 | 설명 |
| --- | --- | --- |
| 학습자 | 교육 참가자 | 세션 코드를 입력하고 워크샵에 참여합니다 |
| 강사 | 퍼실리테이터 | 세션을 생성하고 워크샵을 진행합니다 |
| 관리자 | 운영 담당자 | 교육과정, 강사, 리포트를 관리합니다 |

## 학습자 가이드

> 학습자는 강사에게 받은 **4자리 세션 코드**로 워크샵에 참여합니다.

### 1단계: 워크샵 참여하기

1. 메인 화면에서 **학습자** 카드를 선택합니다
2. **세션 코드** (4자리 숫자)를 입력합니다
3. **이름**을 입력합니다
4. **참여하기** 버튼을 누릅니다
5. 세션 홈 화면으로 이동합니다

> 세션 코드는 강사가 세션 생성 시 자동으로 부여되며, 강사가 화면에 표시하거나 구두로 안내합니다.

### 2단계: 출석 체크 (셀카 + AI 캐릭터 + 자기소개)

세션 홈 화면 상단에 **출석 체크** 카드가 표시됩니다. 강사가 출석을 열면 **출석하기** 버튼이 활성화되며, 출석 체크는 아래 순서로 진행됩니다.

#### Step 1. 출석하기

- **출석하기** 버튼을 누르면 출석 등록이 시작됩니다
- QR 체크인: 강사가 화면에 표시한 QR 코드를 스캔하면 세션 참여와 출석이 한 번에 완료됩니다

#### Step 2. 셀카 촬영

- 카메라 접근 권한을 허용하면 셀카 촬영 화면이 나타납니다
- 원형 셔터 버튼을 눌러 사진을 촬영합니다
- 촬영 후 **다시 찍기** 또는 **이 사진 사용**을 선택합니다
- 셀카를 찍고 싶지 않으면 **건너뛰기**를 눌러 다음 단계로 넘어갑니다

#### Step 3. AI 캐릭터 생성

- 촬영한 셀카를 기반으로 나만의 캐릭터 아바타를 생성합니다
- 3가지 스타일 중 선택할 수 있습니다:
  - 🎨 **카툰** — 귀여운 카툰 캐릭터
  - ✨ **애니메** — 애니메이션 스타일 아바타
  - 👾 **픽셀** — 픽셀 아트 아바타
- **캐릭터 생성** 버튼을 누르면 AI가 캐릭터를 생성합니다
- 원본 셀카와 생성된 캐릭터를 나란히 비교할 수 있습니다
- 마음에 들지 않으면 **다시 생성**, 확정하려면 **이 캐릭터 사용**을 누릅니다

#### Step 4. 키워드 선택

- 4가지 카테고리에서 나를 표현하는 키워드를 **최대 5개** 선택합니다:
  - **성격**: 외향적, 내향적, 꼼꼼한, 창의적, 리더십, 협력적, 낙관적, 분석적, 도전적, 차분한
  - **관심사**: 테크, 디자인, 마케팅, 데이터, AI, 스타트업, 교육, 헬스케어, 금융, 콘텐츠
  - **역할**: 기획자, 개발자, 디자이너, PM, 마케터, 데이터분석가, 리서처, 운영자, 컨설턴트, 교육자
  - **취미**: 운동, 독서, 여행, 게임, 음악, 요리, 사진, 영화, 등산, 명상
- 키워드를 터치하면 선택/해제됩니다 (5개 초과 시 비활성화)

#### Step 5. 자동 자기소개 생성

- 선택한 키워드를 바탕으로 자기소개 문장이 자동 생성됩니다
- 예시: *"안녕하세요, 김철수입니다. 현재 개발자로 활동하고 있습니다. 외향적, 창의적 성격의 소유자입니다. 테크, AI 분야에 관심이 많습니다. 취미는 운동, 음악입니다. 이번 워크샵에서 많은 것을 배우고 싶습니다. 잘 부탁드립니다!"*
- **직접 수정** 버튼으로 자유롭게 내용을 편집할 수 있습니다
- **다시 생성**으로 키워드 기반 소개를 다시 만들 수 있습니다
- **저장하기**를 누르면 출석 + 자기소개 등록이 완료됩니다

> 전체 과정은 약 2~3분 정도 소요됩니다. 셀카와 캐릭터 생성은 건너뛸 수 있습니다.

### 3단계: 주요 기능 사용하기

세션 홈 하단의 **빠른 메뉴** 또는 **하단 내비게이션 바**를 통해 각 기능에 접근합니다.

---

### 기능 목록

#### 홈 (세션 메인)

- 출석 체크 카드 (출석 상태, 참여 인원 확인)
- 빠른 메뉴 (자기소개, 팀, 게시판, 강의후기 바로가기)

#### 자기소개

- 출석 체크 시 생성한 캐릭터 아바타와 키워드 자기소개가 카드로 표시됩니다
- 내 카드에는 "(나)" 표시가 붙습니다
- 선택한 키워드가 `#해시태그` 형태로 표시됩니다
- 다른 참가자의 소개 카드를 볼 수 있습니다
- 자기소개 내용은 언제든 수정할 수 있습니다

#### 팀 구성

- 강사가 배정한 팀 정보를 확인합니다
- 같은 팀 멤버 목록을 볼 수 있습니다

#### 게시판

- 글을 작성하고 다른 참가자의 글을 읽습니다
- 좋아요(하트) 기능으로 공감을 표시합니다
- 정렬: 최신순 / 인기순

#### 점심

- **투표**: 강사가 제시한 식당 후보 중 원하는 곳에 투표합니다
- **주문**: 투표가 종료되면 선정된 식당의 메뉴에서 원하는 메뉴를 주문합니다
- 식당 검색: 카카오맵 연동으로 주변 식당을 검색할 수 있습니다

#### 공지

- 강사가 보낸 공지사항을 확인합니다
- 중요 공지는 상단에 고정됩니다

#### 설문

- 강사가 열어둔 설문에 응답합니다
- 별점(1~5점) 형식으로 만족도를 평가합니다
- 자유 의견을 텍스트로 작성합니다

#### 강의후기

- 워크샵에 대한 후기를 별점과 함께 작성합니다
- 다른 참가자의 후기를 볼 수 있습니다

## 강사(퍼실리테이터) 가이드

> 강사는 세션을 생성하고, 참가자들의 활동을 실시간으로 관리합니다.

### 1단계: 세션 생성하기

1. 메인 화면에서 **강사** 카드를 선택합니다
2. **강사 이름**을 입력합니다
3. **교육과정명**을 입력합니다 (예: 리더십 워크샵)
4. **세션 제목**을 입력합니다 (예: Day 1 - 오리엔테이션)
5. **세션 생성하기** 버튼을 누릅니다
6. 자동으로 4자리 세션 코드가 생성되고 강사 대시보드로 이동합니다

> 생성된 세션 코드를 학습자들에게 안내해주세요. 코드는 사이드바 상단과 대시보드에 항상 표시됩니다.

### 2단계: 대시보드 활용

강사 대시보드에서 한눈에 현황을 파악할 수 있습니다.

| 지표 | 설명 |
| --- | --- |
| 출석 | 현재 출석 인원 수 |
| 게시글 | 작성된 게시글 수 |
| 점심투표 | 투표 참여 수 |
| 후기 | 작성된 후기 수 |
| 설문응답 | 설문 응답 수 |
| 평균만족도 | 설문 평균 점수 |

### 3단계: 기능별 운영 가이드

---

### 출석 관리

- **출석 열기/닫기**: 토글 스위치로 출석을 열고 닫습니다
- **QR 코드**: 자동 생성된 QR 코드와 4자리 세션 코드를 화면에 표시합니다
  - QR 코드를 프로젝터에 띄우면 학습자가 스캔하여 바로 참여할 수 있습니다
- **실시간 목록**: 출석한 참가자 목록이 실시간으로 업데이트됩니다
  - 참가자 이름, 셀카 썸네일, 캐릭터 아바타, 체크인 방식이 표시됩니다
- **진행률 바**: 전체 참가자 대비 출석 인원을 시각적으로 확인합니다

### 자기소개 보기

- 학습자들이 작성한 자기소개 카드를 한눈에 볼 수 있습니다
- 각 카드에 캐릭터 아바타, 이름, 키워드 태그, 자기소개 내용이 표시됩니다

### 팀 구성하기

- **랜덤 팀 배정** 기능으로 참가자를 자동으로 팀에 배정합니다
- 팀 수 설정: 2~20팀 사이에서 원하는 팀 수를 선택합니다
- 배정 결과를 실시간으로 확인할 수 있습니다
- 재배정이 필요하면 다시 랜덤 배정을 실행합니다

### 게시판 관리

- 학습자들이 작성한 게시글을 실시간으로 모니터링합니다
- 게시글 삭제 등 관리 기능을 사용할 수 있습니다

### 공지 보내기

- **공지 작성**: 제목과 내용을 입력하여 학습자에게 실시간으로 전송합니다
- **중요 공지**: 중요 표시를 하면 학습자 공지 목록 상단에 고정됩니다
- 공지 수정 및 삭제가 가능합니다

### 점심 관리

- **식당 투표 열기**: 토글 스위치로 투표를 엽니다
- **식당 후보 등록**: 카카오맵 검색으로 주변 식당을 검색하고 후보에 추가합니다
- **투표 현황 확인**: 각 식당별 투표 수를 실시간으로 확인합니다
- **메뉴 주문 열기**: 투표가 끝나면 주문 기능을 열어 학습자가 메뉴를 선택하게 합니다
- **주문 현황 확인**: 메뉴별 주문 수량을 집계하여 확인합니다

### 후기 보기

- 학습자들이 작성한 강의 후기를 확인합니다
- 별점 평균과 텍스트 후기를 한눈에 볼 수 있습니다

### 설문 관리

- **설문 열기/닫기**: 토글 스위치로 설문 응답을 열고 닫습니다
- **설문 결과**: 각 질문별 평균 점수와 응답 분포를 확인합니다
- **차트**: 시각적인 차트로 결과를 확인할 수 있습니다

## 관리자 가이드

> 관리자는 이메일/비밀번호로 로그인하여 교육과정 전체를 관리합니다.

### 1단계: 로그인

1. 메인 화면에서 **관리자** 카드를 선택합니다
2. 등록된 **이메일**과 **비밀번호**를 입력합니다
3. **로그인** 버튼을 누릅니다
4. 관리자 대시보드로 이동합니다

> 관리자 계정은 Firebase Authentication에 등록된 계정만 사용할 수 있습니다.

### 2단계: 대시보드

관리자 대시보드에서 전체 교육과정 현황을 확인합니다.

| 항목 | 설명 |
| --- | --- |
| 전체 교육과정 | 등록된 교육과정 총 수 |
| 진행 중 | 현재 진행 중인 교육과정 수 |
| 완료 | 완료된 교육과정 수 |

- 최근 교육과정 5개가 목록으로 표시됩니다
- 교육과정을 클릭하면 상세 정보를 확인할 수 있습니다

### 3단계: 교육과정 관리

---

### 교육과정 목록

- 전체 교육과정 목록을 확인합니다
- 상태별 필터: 진행 중 / 완료 / 준비 중
- 새 교육과정을 등록할 수 있습니다

### 교육과정 상세

- 교육과정의 세션 목록을 확인합니다

## 세션 설정 옵션 안내 (강사용)

강사가 세션 진행 중 ON/OFF 할 수 있는 설정입니다.

| 설정 | 기본값 | 설명 |
| --- | --- | --- |
| 출석 (attendanceOpen) | ON | 학습자 출석 체크 허용 |
| 게시판 (boardOpen) | ON | 게시판 글 작성 허용 |
| 자기소개 (introOpen) | ON | 자기소개 카드 작성 허용 |
| 점심 투표 (lunchPollOpen) | OFF | 점심 식당 투표 허용 |
| 후기 (reviewOpen) | ON | 강의 후기 작성 허용 |
| 설문 (surveyOpen) | OFF | 만족도 설문 응답 허용 |

> 각 기능의 토글 스위치를 OFF로 설정하면 학습자 화면에서 해당 기능이 비활성화됩니다.

## 워크샵 진행 시나리오 (추천)

### 시작 전

1. 강사가 **세션 생성** (교육과정명, 세션 제목, 강사 이름 입력)
2. 화면에 **세션 코드**를 공유
3. **출석** 기능 ON 확인

### 오전 세션

1. 학습자들이 세션 코드 입력 또는 **QR 스캔**으로 참여
2. **출석 체크** — 셀카 촬영 → AI 캐릭터 생성 → 키워드 선택 → 자기소개 자동 생성
3. 참가자들의 **자기소개 카드**를 함께 둘러보며 아이스브레이킹
4. **팀 구성** (강사가 랜덤 배정)
5. **게시판**에서 토론 및 의견 공유

### 점심 시간

1. 강사가 **점심 투표 열기**
2. 카카오맵으로 식당 후보 등록
3. 학습자들이 투표
4. 투표 종료 후 **메뉴 주문 열기**
5. 학습자들이 메뉴 선택
6. 강사가 주문 현황 확인 후 주문

### 오후 세션

1. **공지**를 통해 오후 일정 안내
2. **게시판**에서 활동 결과 공유
3. 세션 종료 전 **후기** 작성 요청

### 마무리

1. **설문 열기** (만족도 조사)
2. 학습자들이 설문 응답
3. 설문 결과 확인 및 공유
4. 설문 닫기

## 자주 묻는 질문

### Q. 세션 코드가 만료되나요?

A. 세션 코드는 생성 후 24시간 동안 유효합니다.

### Q. 학습자가 세션 중간에 나갔다가 다시 들어올 수 있나요?

A. 네, 같은 세션 코드와 이름으로 다시 참여할 수 있습니다. 이전 세션 정보가 브라우저에 저장되어 있으면 자동으로 복원됩니다.

### Q. 동시에 여러 세션을 운영할 수 있나요?

A. 각 세션은 고유한 4자리 코드를 가지므로, 여러 강사가 동시에 별도의 세션을 운영할 수 있습니다.

### Q. 관리자 계정은 어떻게 만드나요?

A. 관리자 계정은 Firebase Console에서 Authentication > Users 메뉴에서 이메일/비밀번호 계정을 직접 추가합니다.

### Q. 셀카와 캐릭터 생성은 필수인가요?

A. 아닙니다. 출석 체크 과정에서 **건너뛰기** 버튼으로 셀카 촬영과 캐릭터 생성을 건너뛸 수 있습니다. 키워드 선택만으로도 자기소개가 생성됩니다.

### Q. AI 캐릭터 생성이 안 되면 어떻게 하나요?

A. 네트워크 상태나 서버 상황에 따라 생성이 실패할 수 있습니다. 이 경우 **건너뛰기**를 눌러 키워드 자기소개 단계로 바로 넘어갈 수 있습니다.

### Q. 모바일에서도 사용할 수 있나요?

A. 네, 학습자 화면은 모바일에 최적화되어 있습니다. 강사 화면은 태블릿 또는 노트북 사용을 권장합니다. 셀카 촬영은 모바일 카메라를 사용합니다.

### Q. 데이터는 어디에 저장되나요?

A. 모든 데이터는 Google Firebase (Firestore)에 실시간으로 저장됩니다.
