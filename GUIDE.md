## 역할별 접근 경로

| 역할 | 경로 | 설명 |
|------|------|------|
| **관리자** | `/dashboard` | 코스·퍼실리테이터·보고서 관리 |
| **퍼실리테이터** | `/present` | 세션 진행·실시간 운영 |
| **학습자** | `/join` | 세션 코드로 참여 |

## 1. 관리자 (Admin)

### 로그인
- URL: `/login`
- Firebase Auth 이메일/비밀번호 로그인

### 코스 관리 (`/dashboard/courses`)
1. **코스 생성**: 코스명·날짜·장소 입력 → 저장
2. **세션 추가**: 코스 상세 → "세션 추가" → 세션명·시작시간 설정
3. **참가자 등록**: 세션 상세 → "참가자 추가" → 이름·연락처·역할 입력 (또는 CSV 일괄 등록)
4. **팀 배정**: "팀 자동 배정" 또는 수동 드래그&드롭

### 퍼실리테이터 관리 (`/dashboard/facilitators`)
- 퍼실리테이터 계정 생성 및 코스 권한 부여

### 점심 관리 (`/dashboard/lunch`)
- 식당 즐겨찾기 목록 관리

### 보고서 (`/dashboard/reports`)
- 출석률·만족도·설문 결과 다운로드 (CSV/PDF)

## 2. 퍼실리테이터 (Facilitator)

### 세션 생성 (`/create`)
1. 코스 선택 → 세션 정보 입력
2. **세션 코드 자동 생성** (6자리) — 학습자가 이 코드로 입장
3. QR 코드 자동 생성 → 스크린에 띄우거나 인쇄

### 세션 운영 대시보드 (`/present`)
하단 탭으로 기능 전환:

#### 공지 (`/present/announcements`)
- 실시간 공지 작성 → 학습자 화면에 즉시 표시
- 중요도 레벨 설정 (일반 / 중요 / 긴급)

#### 출석 (`/present/attendance`)
- 참가자 카드 클릭 → 출석 체크 (✓ 표시)
- 전체 출석 현황 실시간 확인

#### 소개 (`/present/intro`)
- 참가자 자기소개 카드 큐레이션
- 캐릭터 이미지 + 한 줄 소개 디스플레이

#### 팀 (`/present/team`)
- **룰렛 배정**: "룰렛 시작" → 팀 자동 랜덤 배정 애니메이션
- 팀 리더 지정, 팀 색상/이름 변경

#### 게시판 (`/present/board`)
- 팀별 칼럼 뷰 / 전체 마소니 뷰 전환
- 포스트 핀 고정, 이미지 업로드 허용/차단 설정

#### 라이브 활동 (`/present/activities`)
활동 유형 3종을 생성·운영한다:

| 유형 | 설명 |
|------|------|
| **투표 (Poll)** | 객관식 선택지 투표. 2개 이상 항목 입력 후 생성. 실시간 막대 차트로 결과 표시 |
| **Q&A** | 학습자가 익명 질문 제출 + 추천. 추천순 정렬로 중요 질문 파악 |
| **워드 클라우드** | 자유 단어 수집. 제출 즉시 크기·위치 애니메이션으로 클라우드 생성 |

**활동 운영 순서**:
1. "+ 활동 추가" → 유형 선택 → 제목·안내문 입력 → 생성
2. 활동 카드의 ▷ 버튼 → **활성화** (Display 화면 자동 전환)
3. ■ 버튼 → 비활성화 (다음 활동으로 넘어갈 때)

**쉬는 시간 타이머**: "타이머" 버튼 클릭 → 분 설정 → 시작 → Display 화면에 카운트다운 표시

#### 점심 (`/present/lunch`)
- 카카오맵 식당 검색 (현재 위치 기반)
- 식당 카드의 🔗 아이콘 → 카카오맵, 🧭 아이콘 → 네이버 지도로 상세 확인
- 식당 후보 등록 → 학습자 투표 → 결과 확인
- 인기 메뉴 항목은 ★ 표시로 강조

#### 설문 (`/present/survey`)
- 실시간 설문 활성화/비활성화
- 응답 현황 차트 실시간 표시

#### 리뷰 (`/present/review`)
- 세션 만족도 평가 수집 (1~5점 + 텍스트)
- 키워드 클라우드 자동 생성

## 3. 학습자 (Learner)

### 참여 방법
1. URL 접속: `https://workshop-management.vercel.app/join`
2. **세션 코드 입력** (퍼실리테이터가 알려준 6자리)
3. 이름 입력 → 입장

### 학습자 화면 (`/session`)

#### 소개 (`/session/intro`)
- 캐릭터 선택 (동물 아바타 10종)
- 한 줄 자기소개 입력 → 즉시 스크린 표시

#### 출석 (`/session/attendance`)
- 본인 출석 확인

#### 팀 (`/session/team`)
- 배정된 팀 확인, 팀원 목록 보기

#### 게시판 (`/session/board`)
- 포스트 작성 (텍스트 + 이미지)
- 다른 팀원 포스트에 좋아요·댓글

#### 라이브 활동 (`/session/activities`)
퍼실리테이터가 활성화한 활동에 참여:
- **투표**: 선택지 중 하나 탭 → 실시간 결과 확인
- **Q&A**: 질문 입력 → 제출 · 다른 질문에 ♥ 추천
- **워드 클라우드**: 단어 입력 → 즉시 화면에 등장

#### 점심 (`/session/lunch`)
- 식당 후보 투표 (1인 1표)
- 카카오맵(🔗) 또는 네이버 지도(🧭) 링크로 위치 확인

#### 설문 (`/session/survey`)
- 퍼실리테이터가 활성화한 설문 응답

#### 리뷰 (`/session/review`)
- 세션 만족도 별점 + 자유 텍스트 입력

## 4. Display 모드 (프로젝터 화면)

### 접속
- URL: `/display?sessionId={세션ID}`
- 또는 퍼실리테이터 화면에서 "Display 모드 열기" 클릭

### 표시 기능
퍼실리테이터가 활성화한 기능이 자동으로 전환 표시됨:

| 기능 | 화면 내용 |
|------|----------|
| **환영 화면** | 세션 코드 + QR 코드 (참가자 입장용) |
| **공지** | 공지 내용 풀스크린 |
| **출석** | 실시간 출석 현황 |
| **소개** | 참가자 캐릭터 카드 그리드 |
| **팀** | 팀별 구성원 카드 |
| **게시판** | 팀 칼럼 실시간 업데이트 |
| **투표** | 실시간 막대 차트 |
| **워드 클라우드** | 단어 팝인 애니메이션 |
| **Q&A** | 질문 목록 + 추천수 |
| **타이머** | 쉬는 시간 카운트다운 |

> **팁**: 프로젝터에 Chrome 전체화면(F11)으로 띄우면 최적

## 6. 자주 묻는 질문

**Q. 학습자가 QR 스캔 없이 접속하려면?**
→ `https://workshop-management.vercel.app/join` 접속 후 세션 코드 직접 입력

**Q. 세션 코드를 분실했을 때?**
→ `/present` 화면 상단에 세션 코드 항상 표시됨

**Q. 게시판 이미지가 업로드가 안 될 때?**
→ Firebase Storage 규칙 확인 또는 이미지 크기 5MB 이하로 압축

**Q. Display 화면이 자동 전환되지 않을 때?**
→ 브라우저 탭 활성 상태 확인 (백그라운드 탭은 Firestore 실시간 리스너가 느려질 수 있음)

**Q. 팀 룰렛 결과를 초기화하려면?**
→ 팀 탭 → "팀 초기화" 버튼 → 다시 룰렛 실행
